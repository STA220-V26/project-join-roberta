Sociodemographical factors predicting healthcare utilization and insurance coverage patterns

Author

Lovisa Johansson & Roberta Merialdo

Note

This report was generated using synthetic data! It is only used for demonstration purposes and does not reflect real patients or healthcare providers. The contents have not been reviewed or validated by medical professionals.

Introduction

Understanding how sociodemographic factors influence healthcare utilization and insurance coverage is essential for identifying disparities in healthcare access and outcomes. Age, gender, and race are known to be associated with differences in health needs and healthcare-seeking behaviour. This project explores these relationships using a synthetic healthcare dataset, with the aim of identifying patterns that may reflect real-world inequalities in healthcare systems.

Research Question

“Do sociodemographic factors (age, gender, race) predict healthcare utilization and insurance coverage patterns?”

Data and Methods

Dataset Description

This analysis uses synthetic healthcare data generated by Synthea, a synthetic patient population simulator. The dataset contains 17 interlinked CSV files covering patients, clinical encounters, conditions, medications, procedures, and insurance information. The primary datasets used in this analysis are:

  • patients — demographic information including age, gender, race, and insurance coverage
  • encounters — records of all clinical visits per patient, used to measure healthcare utilization
  • payer_transitions — records of insurance coverage changes over time

Variable Definitions

Variable Source Description
age patients Age in years, derived from birthdate
gender patients Patient gender (M/F)
race patients Patient race (white, black, asian, etc.)
n_encounters encounters Number of clinical encounters per patient (outcome 1)
healthcare_coverage patients Total insurance coverage amount in USD (outcome 2)

Statistical Methods

Two outcomes were analysed:

  1. Healthcare utilization (n_encounters) — a count variable modelled using Poisson regression
  2. Insurance coverage (healthcare_coverage) — a continuous variable modelled using linear regression

Prior to regression, hypothesis tests were conducted for each predictor:

  • Age — Spearman correlation (non-parametric, due to skewed distributions)
  • Gender — Independent samples t-test
  • Race — Kruskal-Wallis test (non-parametric, multiple groups)

# Calculate age from birthdate
dt_patients[, age := as.numeric(difftime(Sys.Date(), as.Date(birthdate), units = "days")) / 365.25]

# Count number of encounters per patient (healthcare utilization)
encounter_counts <- dt_encounters[, .(n_encounters = .N), by = patient]

# Join patients with encounter counts and coverage
analysis <- dt_patients[encounter_counts, on = .(id = patient), nomatch = 0]
analysis <- analysis[, .(id, age, gender, race, n_encounters, healthcare_coverage)]

# Remove deceased patients
analysis <- analysis[is.na(dt_patients[match(analysis$id, dt_patients$id), deathdate]) |
                     dt_patients[match(analysis$id, dt_patients$id), deathdate] == ""]

Discussion and Conclusion

Key Findings

The analysis examined whether age, gender, and race predict healthcare utilization (number of encounters) and insurance coverage among synthetic patients. Results from the hypothesis tests and regression models reveal the extent to which these sociodemographic factors are associated with differences in healthcare access and use.

Clinical Implications

If sociodemographic factors such as race and gender are significantly associated with healthcare utilization and coverage, this may indicate systemic disparities in healthcare access. Such findings — even in synthetic data — can motivate the development of more equitable healthcare policies and targeted interventions for underserved groups.

Limitations

  • The data is entirely synthetic and generated by Synthea. While it mimics real-world patterns, it does not reflect actual patients or healthcare providers, and conclusions cannot be generalised to real populations.
  • The analysis does not account for confounding variables such as income, geographic location, or underlying health conditions, which may explain part of the observed associations.
  • The Poisson regression assumes that the mean equals the variance in encounter counts. If overdispersion is present, a negative binomial model would be more appropriate.

Conclusion

This project explored the relationship between sociodemographic factors and healthcare outcomes using a reproducible targets-based pipeline. The findings provide a foundation for further investigation into healthcare disparities and demonstrate the value of structured, reproducible data analysis workflows.
