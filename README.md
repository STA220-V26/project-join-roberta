# Sociodemographic Factors and Healthcare Utilization

**Authors:** Lovisa Johansson & Roberta Merialdo
**Course:** STA220

## Research Question

> Do sociodemographic factors (age, gender, race) predict healthcare utilization and insurance coverage patterns?

## Data

This project uses synthetic healthcare data generated by [Synthea](https://github.com/synthetichealth/synthea), containing 17 CSV files covering patients, encounters, conditions, medications, procedures, and more. The data is automatically downloaded when running the pipeline.

## How to Run

1. Clone the repository:
   ```
   git clone https://github.com/STA220-V26/project-join-roberta.git
   ```
2. Open the folder in RStudio or Positron
3. Set the working directory to the project folder
4. Run the pipeline in R:
   ```r
   library(targets)
   tar_make()
   ```
   This will automatically install required packages, download and unzip the data, run the analysis, and render `report.html`.

## Repository Structure

```
├── R/
│   ├── get_codebook.R      # Scrapes variable descriptions from Synthea wiki
│   └── data_scans.R        # Generates pointblank data quality reports
├── data-fixed/             # CSV data files (auto-generated, not tracked in git)
├── output/                 # Output folder
├── report.qmd              # Quarto report source
├── report.html             # Rendered report
└── _targets.R              # Targets pipeline definition
```

## Results

The rendered report (`report.html`) contains:
- Exploratory data analysis with figures and tables
- Hypothesis tests for age, gender, and race
- Poisson regression for healthcare utilization
- Linear regression for insurance coverage
